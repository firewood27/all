# 맥북 키패드와 아이콘

- ⌥: options
- ⌘: command
- ⇧: shift
- ⇪: capslock
- ⌃: control
- ⇥: tab
- ⎋: ESC

# 맥북 키배열과 윈도우 키가 있는 106키보드의 키배열

1. 맥북: fn ⌃ ⌥ ⌘ 
2. 106키 : ⌃ ⌘ ⌥ fn

# Intellij keymap

- settings: ⌃ ⌘ S
- New : ⌃ enter
- Navigator: ⌘ 1

# 맥북 단축키
